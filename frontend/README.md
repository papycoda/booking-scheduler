# Booking Scheduler Frontend

Next.js 14 App Router frontend for the booking scheduler spec.

## Setup

```powershell
npm.cmd --prefix frontend install
```

## Run

```powershell
npm.cmd --prefix frontend run dev
```

## Environment

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000/api/v1
```

In this workspace, dependency installation has repeatedly timed out before creating `frontend/node_modules` or `frontend/package-lock.json`, so the frontend build has not yet been verified here.
